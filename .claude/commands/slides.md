---
description: 세미나 발표용 PPT 슬라이드 구성 (아웃라인 우선, 확인 후 제작)
---

이 프로젝트는 PQC 세미나 발표 자료 제작용이다. 사용자가 지정한 회차 또는 주제(`$ARGUMENTS`)로 아래 순서를 따른다:

1. 소스 확인: `Notes/<회차>.md`, 해당 `Lecture slides/` 원본, `Papers/A Gentle Introduction to Lattice-Based Cryptography.pdf`에서 관련 내용을 모은다. 부족하면 사용자에게 추가로 강조하고 싶은 포인트를 먼저 묻는다.
2. 슬라이드를 만들기 전에 **아웃라인부터** 제시한다 (슬라이드별 제목 + 핵심 bullet 1~2줄). 전체 분량, 발표 시간, 청중 수준(세미나 팀원 대상인지 외부 발표인지)을 확인한다.
3. 아웃라인에 대해 사용자 승인/수정을 받는다. 임의로 바로 pptx를 만들지 않는다.
4. 승인되면 pptx 스킬을 사용해 실제 슬라이드 파일을 만든다. 디자인은 과한 장식 없이 텍스트 가독성과 수식/그림 배치를 우선한다. 원본 강의 슬라이드의 표기법을 그대로 따르되, errata(알려진 오류)가 있다면 교재 기준으로 수정해서 반영한다.
5. 결과물은 `presentation/` 폴더에 `<회차>_발표자료_초안.pptx` 형태로 저장한다.
6. 완성 후 발표 리허설용으로 각 슬라이드 발표 스크립트(간단한 노트)를 원하면 추가로 `presentation/`에 저장한다.

목표는 세미나 팀원들이 이해하기 쉬운 발표 자료 — 화려함보다 논리적 흐름과 핵심 강조가 우선.
